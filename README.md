## Arquitetura MVVM com react

<p>
    Com o MVVM no react, é possível separa a logica de renderização do componente/tela da view em si, isso permite uma melhor manutenção do código
</p>

<h2>
    Modelo
</h2>

```bash
    -- Form
        -- index.tsx //ViewModel
        -- model.tsx //Model
        -- view.tsx //View
```
